Project CV Generator

项目简历生成器：7 个 Python 机械臂先行提取确定性数据（技术栈/技术亮点/测试质量/功能模块/技术挑战/贡献证据/工程方法论），AI 做业务价值映射与角色确认，产出 8 段式中文面试级项目简历 + 一页版 + 面试背诵特训包。支持 Java/PHP/Python/Go/Node/Vue/React，零幻觉。

执行全流程（10 步 Workflow）

01 技术栈自动检测
脚本
project_scanner.py 检测语言/框架/规模，告知结果并询问特殊关注点。
02 技术亮点扫描（代码+文档）
脚本
tech_highlight_scanner.py 识别模式/缓存/异步/MQ 等亮点。
03 工程质量评估
脚本
test_quality_scanner.py 评估测试/CI/Docker 成熟度。
04 业务模块识别
脚本
feature_module_scanner.py 识别 API/数据模型等业务模块。
05 技术挑战提取
脚本
challenge_extractor.py 从 TODO/优化/FIXME 提取攻克过的难题。
06 贡献证据提取（git 改动热点）新增
脚本
contribution_scanner.py 扫描 git 改动热点，量化个人贡献证据（无 git 时优雅降级）。
07 工程方法论提炼（16 类方法论）新增
脚本
methodology_extractor.py 从代码/文档识别 DDD/分层/设计模式等工程方法论与思维。
08 AI 业务价值映射
参考
references/business-value-guide.md 将技术指标映射为业务价值（面试官记忆线）。
09 交互确认角色与贡献边界
参考
references/fuzzy-expression-guide.md 模糊量级表达 + 角色/职级/边界必须由用户确认。
10 AI 填列 + 组装三产物
动作
基于 7 个机械臂 JSON，AI 填语义描述，产出 3 份产物并写入被分析项目 docs/。
产出
① 8 段式项目简历（含「工程方法论与思维」）② 一页简历版 ③ 面试特训包（电梯演讲 + Top3 + STAR + 记忆卡）。
红线
脚本先行、数字真实、零幻觉（无数据不编造）、仅中文、三产物随项目走纳入版本管理。
