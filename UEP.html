工程协议层

UEP

通用工程协议（UEP v2.0.0），用于自动化项目管理和多 Agent 协作：协调同一代码库上的多个 AI Agent，通过不可变日志跟踪任务进度，自动合并并发交付物（路由、依赖、环境变量）。以 Python 脚本作确定性机械臂（scripts/*.py）+ Skill 文件作人类可读包装器。

关键信息

触发条件

  • 新项目启动："启动新项目"、"初始化项目"
  • 多 Agent 协作："多Agent协作"、"并行开发"
  • 遗留项目重构："重构前快照"、"项目状态分析"
  • Bug 修复追踪："修复Bug"、"追踪问题"

排除边界

  • 单 Agent 简单任务
  • 纯代码生成（无并行/无追踪需求）
  • 临时性任务
  • 小型项目（<5 文件）

产出物

  • STATE.json：任务队列、API 清单、文件索引（.ai-workflow/uep/）
  • TASKS/：Markdown 任务文档
  • LOGS/：JSON 交付报告（.ai-workflow/uep/LOGS/）
  • FINAL_REPORT.md：自动拼装合并结果
