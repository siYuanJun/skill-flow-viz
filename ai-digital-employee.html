AI Digital Employee · 数字员工平台

关键信息

触发条件

  • 数字员工 / AI 员工 / 智能体平台建设
  • 多智能体协作、Agent 技能市场设计
  • 受监管行业（金融·央企·政企·法律）的合规适配与私有化部署

排除边界

  • 纯 RPA（无 AI 决策能力）
  • 简单聊天机器人（无自主任务执行）
  • 通用 LLM 提示词工程（无关 Agent 系统）
  • 纯 UI 前端设计（无后端 Agent 基建）
  • 一次性脚本类非 AI 工作流工具

产出物（7 类具体交付物）

  • PRD → 架构图 → 技术栈矩阵
  • 行业清单 → 路线图（POC→Scale→Ecosystem）
  • Agent 技能目录 → 评估看板
  • 原则：文件落盘 + 清单确认才算完成

四大运行模式

  • Scaffold 单 Agent 原型
  • Production 多 Agent 受治理
  • Library 模板 / 清单库
  • Governed 金融·央企·政企（信创 + 私有化 + 全审计）
