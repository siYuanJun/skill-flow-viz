执行全流程（设计驱动，非代码生成）

01 读八大设计原则（开发前必读）
内容
四态模型(Write/Select/Compress/Isolate)、Token 预算、分层记忆、压缩矩阵、防幻觉、回调体系、可观测性、持久化恢复。
02 设计上下文四态架构
动作
规划 Write（持久化到窗口外）/ Select（语义检索相关）/ Compress（超阈值压缩）/ Isolate（多任务隔离）。
03 落地分层记忆 + Token 预算
动作
Layer1 短期内存 / Layer2 中期缓存(24h TTL) / Layer3 长期向量库；token_budget.py 三道防线：70%告警 / 85%自动压缩 / 95%强截断。
04 接入机械臂脚本
脚本
token_budget / context_compressor / context_audit（实体对比+信息丢失）/ session_manager（持久化恢复）。
05 过生产检查清单 + 产出文档
产出
.ai-workflow/rag-context-engine/docs/ 设计文档 + sessions/ 存档 + metrics/ 监控数据。
红线
防幻觉三防护：压缩前后实体对比 / 置信度标记 / 疑问回退检索。
